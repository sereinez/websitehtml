# Leafy Haven

Лендинг-страница для магазина комнатных растений «Leafy Haven». Адаптивный одностраничный сайт с секциями каталога, преимуществ, советов по уходу и формой обратной связи.

## Структура проекта

```
project/
├── index.html       # Основная разметка страницы
├── style.css         # Стили (включая адаптивные медиа-запросы)
└── images/           # Изображения растений и баннеров
    ├── photo1.png
    ├── photo3.png
    ├── photo4.png
    ├── photo6.png
    ├── photo7.png
    └── photo8.png
```

## Технологии

- HTML5
- CSS3 (Flexbox, Grid, медиа-запросы)
- Tailwind CSS (через CDN, используется для отдельных секций)
- Google Fonts — шрифт **Syne**

## Описание секций страницы

| Секция | Описание |
|---|---|
| Навигация (`bubble-container`) | Верхнее меню с ссылками на разделы сайта |
| Главный блок (`main-content-container`) | Заголовок, описание и кнопки призыва к действию |
| Каталог растений (`page-2`) | Галерея из карточек с фото и названиями растений |
| Баннер (`page-3`) | Полноэкранное изображение с заголовком |
| Этапы ухода (`page-4`) | Пошаговая инструкция по выбору и уходу за растением |
| Преимущества (`page-5`) | Блок с иконками и описанием плюсов комнатных растений |
| Призыв к действию (`page-6`) | Блок с приглашением присоединиться и доп. преимуществами |
| Подвал (`footer`) | Контактная информация, ссылки и копирайт |

## Адаптивность

Сайт адаптирован под разные размеры экранов с помощью медиа-запросов:

- **до 1300px** — сужение десктопных блоков под доступную ширину контейнера
- **до 1024px** — перестройка сетки галереи и блоков преимуществ
- **до 599px** — мобильная версия: одна колонка, уменьшенные шрифты, вертикальное расположение элементов

## Как запустить

1. Скачайте/клонируйте папку проекта со всеми файлами и подпапкой `images`.
2. Откройте `index.html` в браузере.

Никакой установки или сборки не требуется — сайт состоит из статичных HTML/CSS файлов.

## Известные ограничения

- Кнопки и ссылки в навигации и подвале не имеют активных переходов (`href`) — добавьте их при необходимости.
- Изображения подключаются по относительным путям из папки `images/` — убедитесь, что названия файлов совпадают.
